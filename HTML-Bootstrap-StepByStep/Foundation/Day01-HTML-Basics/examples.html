HTML5 Document Structure

Every HTML5 document should include:

DOCTYPE Declaration: <!DOCTYPE html> tells the browser this is HTML5

HTML Element: The root element with language attribute

Head Section: Contains metadata and document information

Body Section: Contains all visible content

Text Formatting Examples

Importance and Emphasis

Use strong for important content and emphasis for stressed text.

Visual formatting: bold text and italic text without semantic meaning.

Other Text Elements

Here are various text formatting options:

Underlined text for stylistic purposes

Strikethrough text for deleted content

Highlighted text for relevance

Small print text for fine print

Chemical formula: H2O (subscript)

Mathematical expression: E=mc2 (superscript)
