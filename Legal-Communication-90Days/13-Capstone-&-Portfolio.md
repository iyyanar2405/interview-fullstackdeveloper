# 13 — Capstone & Portfolio (Week 13)

Theme: Assemble and present your best work.

Daily plan
- Day 1: Select artifacts (emails, memos, clauses, notices)
- Day 2: Edit for clarity, tone, and citations
- Day 3: Create a portfolio README with context and excerpts
- Day 4: Record a 5–7 minute capstone talk
- Day 5: Peer review and iterate
- Day 6: Finalize and share portfolio
- Day 7: Reflect and set next 90-day goals

Deliverables
- Portfolio with 10–15 artifacts; 1 capstone talk; reflection note
